# 🎮 Keyboard Trainer

Practica y optimiza tu layout de teclado split con este trainer interactivo estilo cyberpunk.

![React](https://img.shields.io/badge/React-19-blue?logo=react)
![Vite](https://img.shields.io/badge/Vite-8-purple?logo=vite)
![TypeScript](https://img.shields.io/badge/TypeScript-5.0-blue?logo=typescript)

## ✨ Características

- 🎨 **Diseño Cyberpunk**: Estilo neón con animaciones y efectos visuales
- ⌨️ **Múltiples Teclados**: Soporte para Corne, Sofle, Lily58, Ferris, Kyria y más
- 📊 **Sistema de Progreso**: XP, niveles y logros desbloqueables
- 🏆 **27 Logros**: Desde rachas de 10 hasta 200 palabras
- 🎮 **4 Modos de Juego**: Práctica, Contrarreloj, Supervivencia y Zen
- 🔊 **Efectos de Sonido**: Sintetizados con Web Audio API
- ✨ **Guía Visual**: Teclas se iluminan mostrando qué presionar y errores

## 🚀 Demo

[Ver demo en vivo](https://keyboard-trainer.vercel.app)

## 🛠️ Stack

- **Vite** + **React** 19 + **TypeScript**
- **Vitest** para testing
- **ESLint** para calidad de código
- LocalStorage para persistencia de datos

## 📦 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/madkoding/keyboard-trainer.git
cd keyboard-trainer

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Build de producción
npm run build
```

## 🎮 Uso

1. Selecciona tu modelo de teclado (Corne, Sofle, Lily58, etc.)
2. Elige un modo de juego
3. Escribe las palabras que aparecen en pantalla
4. Observa cómo las teclas se iluminan para guiarte
5. ¡Desbloquea logros y sube de nivel!
